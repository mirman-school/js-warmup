FizzBuzz Applied

Objective

Write FizzBuzz. In FizzBuzz, a function fizzbuzz() takes an argument n and does the following:

  • If divisble by 3, returns "Fizz"
  • If divisble by 5, returns "Buzz"
  • If divisble by 3 and 5, returns "FizzBuzz"
  • Otherwise, return null

That's FizzBuzz. But that has nothing to do with The DOM, so we're not finished.

Below, you will find 3 <div> elements, each with a unique <ul> for results. Each <ul> has a specific ID we can use to "capture" it with JavaScript.

So after writing a working fizzbuzz(), your next job is to take the results from 1 to 50 and append them to the appropriate list.
